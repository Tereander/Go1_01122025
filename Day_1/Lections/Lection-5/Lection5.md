## Лекция 5. Условный оператор

**Шаблоны:**  
Минимальный 
```
if condition {
    // body
}
```
С блоком `else`:  

```
if condition {
    // body 1
} else {
    // body 2
}
```

С блоками`else if`:  

```
if condition 1 {
    // body 1
} else if conditon 2 {
    // body 2
} else if condition 3 {
    // body 3
} else {
    // body 4
}
```
